# Project Sentinel: OT Network Monitoring with Zeek & Splunk

## Overview

Project Sentinel is a hands-on cybersecurity project focused on monitoring Industrial Control System (ICS) and Operational Technology (OT) network traffic using Zeek and Splunk.

The objective is to simulate how a Security Operations Center (SOC) can monitor an industrial environment, collect network telemetry, detect suspicious activity, and investigate potential security incidents.

---

## Technologies

- Zeek
- Splunk Enterprise
- Ubuntu (WSL2)
- Wireshark
- Git & GitHub

---

## Project Goals

- Monitor industrial network traffic
- Analyze ICS protocols
- Generate Zeek logs
- Ingest logs into Splunk
- Build SOC dashboards
- Detect suspicious activity
- Produce incident reports
