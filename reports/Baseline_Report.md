# Baseline Assessment Report

## Executive Summary

This report documents the initial baseline assessment performed as part of Project Sentinel.

The objective is to understand normal communications within the fictional NorthRiver Timber Manufacturing OT environment before developing detection logic and SOC dashboards.

Status: In Progress

- Document evidence-based findings
