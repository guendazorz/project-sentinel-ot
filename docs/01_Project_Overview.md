# Project Sentinel: OT Security Monitoring Platform

## Executive Summary

Project Sentinel is a portfolio cybersecurity project focused on designing and validating an Operational Technology (OT) security monitoring workflow using Zeek and Splunk.

The project simulates a fictional manufacturing company, NorthRiver Timber Manufacturing, where a Junior OT Security Analyst is responsible for improving visibility into industrial network traffic, identifying assets, detecting suspicious communications, and documenting findings.

## Objectives

- Build an OT network monitoring workflow
- Analyze industrial packet captures with Zeek
- Identify hosts, services, and communication patterns
- Establish a network baseline
- Ingest Zeek logs into Splunk
- Build SOC-style dashboards and detections
- Produce evidence-based analyst reports

## Analyst Role

**Role:** Junior OT Security Analyst

Responsibilities include:

- Deploying monitoring tooling
- Reviewing Zeek network logs
- Identifying OT assets and protocols
- Investigating suspicious traffic
- Creating dashboards and detection logic
- Writing recommendations for improved visibility

## Tools Used

| Tool | Purpose |
|---|---|
| Zeek | Network traffic analysis and log generation |
| Splunk Enterprise | SIEM ingestion, search, dashboards, and alerts |
| Ubuntu on WSL2 | Linux analysis environment |
| VS Code | Documentation and project development |
| Git / GitHub | Version control and portfolio hosting |
| Wireshark | Packet-level validation |
